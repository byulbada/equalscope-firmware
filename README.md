# Equalscope — 펌웨어 & 사용자 매뉴얼

프리다이빙 이퀄라이징 훈련 측정 장비 **Equalscope**의 펌웨어 배포 및 사용자 매뉴얼 저장소입니다.

> 장비 소스 코드는 비공개입니다. 이 저장소에는 **배포용 펌웨어(.bin)와 문서만** 포함됩니다.

## 📥 펌웨어 다운로드 / 업데이트

1. [Releases](../../releases) 에서 최신 펌웨어 파일(`equpdate_<버전>.bin`)을 내려받습니다.
2. SD 카드 맨 위(루트) 폴더에 복사합니다.
3. SD를 기기에 꽂고 **전원을 켭니다** → 부팅 중 자동 인식 → 업데이트 → 재시작.
4. 업데이트가 끝나면 SD의 펌웨어 파일은 **자동으로 삭제**됩니다.

> - SD에는 펌웨어 파일을 **하나만** 올려 두는 것을 권장합니다.
> - 현재 펌웨어 버전은 기기의 **설정 → 도움말** 화면 하단에서 확인할 수 있습니다.
