이퀄라이징은 눈에 보이지 않습니다.

강사는 자세·표정·말로만 상태를 추정하고, 교육생은 "잘 되고 있는지, 너무 세게 밀고 있는지, 리듬이 일정한지"를 감각에만 의존합니다. Equalscope는 이 보이지 않는 압력을 그래프로 보여줍니다.

WHY EQUALSCOPE

장비 하나로 끝나는 이퀄라이징 훈련

스마트폰·PC 없이, 켜자마자 바로. 측정부터 분석·기록·비교까지 장비 안에서 완결됩니다.

1

스마트폰 없이 단독 동작

앱 실행, 블루투스 연결, 화면 잠금, 알림 같은 변수 없이 수업 흐름이 끊기지 않습니다. 켜면 바로 측정.

2

초당 40회, 부드러운 실시간 그래프

40Hz 고속 측정으로 끊김 없이 매끄럽게. 장비가 직접 측정·표시하므로 통신 지연 없이 부드럽게 이어집니다.

3

훈련 후 자동 분석·조언

반복 횟수·주기·상승/하강 속도·정점 유지 시간을 요약하고, 압력 패턴으로 기술 유형을 추정합니다.

FEATURES

주요 기능

이퀄라이징 훈련을 더 편하고 정확하게 만들어 주는 기능들입니다.
단독 동작부터 메트로놈·스톱워치·눈금 밸브까지.

실시간 압력 그래프

40Hz(초당 40회) 고속 측정으로 끊김 없는 부드러운 곡선. 통신 지연 없이 장비가 바로 그려, 빠른 이퀄 반복도 놓치지 않습니다.

최고/최저 압력 알람

설정 범위를 벗어나면 소리로 알림 — 화면을 계속 보지 않아도 됩니다.

메트로놈 · 스톱워치

일정한 리듬으로 반복 훈련. 긴 간격은 CO2 테이블 같은 호흡 훈련에도 활용.

3점 터치 스크린샷

화면을 세 손가락으로 터치하면 캡처되어 SD에 저장 — 교육생 피드백 자료로.

측정 저장 · 참조 비교

측정은 CSV로 자동 저장(SD 우선, 없으면 내부 메모리). 좋은 샘플은 참조 그래프로 겹쳐 비교.

자동 분석 · 조언

반복 통계 요약 + 압력 패턴 기반 기술 추정. 감각이 아닌 숫자로 훈련을 돌아봅니다.

압력 조절 밸브 (눈금)

눈금이 있는 압력 조절 다이얼. 자주 쓰는 설정값을 정확히 반복할 수 있습니다. 입 안 공기가 줄어드는 수심을 지상에서 시뮬레이션(특히 마우스필).

멀티 테마 · 한/영 · 펌웨어 업데이트

여러 색상 테마와 한국어·영어 지원. SD 카드에 파일만 넣으면 펌웨어 자동 업데이트 — PC·앱 없이도 기능이 계속 발전합니다.

HOW TO USE

꺼내서 쓱 연결하고 바로 측정

꺼내 쉽게 연결하고 바로 측정 — 분석까지 세 단계, 영상으로도 확인하세요.

1

연결

본체에 튜브를 딸깍 끼우고, 튜브 끝에 이퀄벤트를 연결합니다.

2

측정

시작 버튼을 누르고 이퀄라이징. 압력이 실시간 그래프로 그려집니다.

3

분석

측정이 끝나면 자동 저장. 분석 화면에서 통계와 조언을 확인합니다.
